Fluid UI

Deprecation Notice

The current Fluid UI codebase is deprecated and will no longer receive any feature updates. Critical security and bugfix updates may still be made available when needed.

Because of this, the existing codebase has moved to the jottocraft/fluid-old GitHub repository, and code that previously wasn't published has been added. Additionally, the entire project, including all 150+ changes made within its lifetime, is now freely available in the public domain as-is under the unlicense.

If you have any questions or concerns about the future of Fluid UI, please contact me at hello@jottocraft.com.
