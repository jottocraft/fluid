Integrated dark mode


Websites that use a dark theme usually don't get much respect. Custom CSS and JavaScript are usually required to get a dark mode. Teleport Fluid has this built in out of the box, and super easy to use.


1

Toggle dark mode with JavaScript

fluid.dark();



2

Enable dark mode with CSS by adding the dark class to the body

<body class="dark"></body>
