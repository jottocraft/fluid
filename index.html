Integrated dark mode


Websites that use a dark theme usually don't get much respect. Custom CSS and JavaScript are usually required to get a dark mode. Teleport Fluid has this built in out of the box, and super easy to use. Everything on this page supports dark mode, and will respond to this demo, so try it out with the other demos to see how powerful it is!


1

Toggle dark mode with JavaScript

fluid.dark();



2

Enable dark mode with CSS by adding the dark class to the body

<body class="dark">


Navigation bar


Setup a navigation bar easily: 

          
<nav class="right">
    <div class="nav-wrapper">
        <ul>
            <li>
            <a href="#"><i class="material-icons left">info</i></a>
            </li>

            <li>
            <a href="#"><i class="material-icons left">help</i></a>
            </li>

            <li>
            <a href="#"><i class="material-icons left">build</i></a>
            </li>
        </ul>
    </div>
</nav>

You can also move it to the left by changing its class: 


<!-- Left alignment -->
<nav class="left">


Easy to use loader


Setup a loader without having to run initialization code:

<loader></loader>

Show the loader (The loader cannot have any display style overrides. Use the hidden class instead to hide by default)

fluid.load(true);

Hide the loader

fluid.load(flase);

Card UI


If you don't nessisarily need a full page for content, but you don't want it being part of the page itself, make it a card


  <div id="democard" class="card">
  <!-- Card Content -->
  </div>

Add the "close" class to make the card hidden by default

<div id="democard" class="card close">

Add the "focus" class to make the card float at the top of the page, scroll the user to the top, and blur the background. Clicking on the background will also close the card!

<div id="democard" class="card focus close">

Because focus cards blur the background and close the card on click, you will need to create a content div as the parent for every part of your page, excluding the focus card itself (It doesn't matter where the card is)

<div class="content">

To show a card:

fluid.cards(card-id)

To close all cards:

fluid.cards.close();

To close only focus cards:

fluid.cards.close.focus();

Other


Everything else that isn't specified here can be found at Materialize's documentation. Fluid CSS is built off of Materialize 0.100.2.


Additional notes about Materialize

Most things should work as expected from Materialize's code. However, some things may be unstable, or not work altogether. Note that Fluid is still in beta, so this may be inaccurate. Email jotto3053@gmail.com with any issues you think should be added to this list.


Unsupported


  • Materialize input labels
  • Textareas (Will be ready by the final release)

Buggy


  • Materialize input icon prefixes
  • Materialize input messages
  • Materialize input character counter
