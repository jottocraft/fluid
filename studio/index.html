Project-based

Fluid Studio is based on projects, not files and folders. If you open a folder with a valid package.json, Fluid Studio will show the version number and full name of the project.
If your folder contains a file named a png file that matches the name of the folder, project name, or is just icon.png, Fluid Studio will also display your project's icon.
Fluid Studio will also detect if you have different versions of your project for different platforms and will display a platform switcher (web, mobile, and desktop are currently supported).
widgets

Fluid UI Integration

Fluid Studio was built with the latest Fluid UI v3 release. You can easily insert Fluid UI elements by right clicking and selecting "Insert Fluid UI Element", or pressing Control + M. This brings up a menu letting you select an element, or even include code to load Fluid UI, with just one click.
Fluid UI's JavaScript functions are also pre-programmed into Autofill and Fluid UI's latest files can also be added to your website with just one click in Fluid Studio (via the Fluid UI Elements menu).
cloud

Git integration

If the current folder is set up with Git and you are logged in with git, a git bar will appear near the bottom-right with the amount of changes you've made and the branch you are on.
You can easily pull, commit, and push directly from Fluid Studio. The project selector will also show your Git name and GitHub profile picture, and will let you clone / open your repos in addition to recent folders.
